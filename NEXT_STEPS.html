Swerte3 - What To Do Next

Project path: C:\Users\jonel\OneDrive\Desktop\Jonel_Projects\Swerte3

1) Configure environment files

  1. Backend: copy backend/.env.example to backend/.env.
  2. Fill required values:
    • DATABASE_URL (PostgreSQL connection)
    • SECRET_KEY
    • ADMIN_API_KEY (required to trigger ingest endpoint)
    • LLM_API_KEY (required for premium Miro + swarm)
    • PAYMONGO_SECRET_KEY and PAYMONGO_WEBHOOK_SECRET (for payments)
  3. Mobile: copy mobile/.env.example to mobile/.env and set EXPO_PUBLIC_API_URL.

2) Start backend API

cd backend
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
alembic upgrade head
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

Check: open http://127.0.0.1:8000/health and confirm {"status":"ok"}.

3) Ingest Google Sheets data (9AM / 4PM / 9PM)

After backend is running and ADMIN_API_KEY is set:

curl -X POST "http://127.0.0.1:8000/api/internal/ingest" ^
  -H "X-Admin-Key: <YOUR_ADMIN_API_KEY>"

Expected: JSON containing inserted, skipped, and possible errors.

4) Test auth flow

  1. Call /api/auth/otp/request with phone.
  2. When SMS_PROVIDER=console, OTP appears in backend logs.
  3. Verify with /api/auth/otp/verify to receive JWT tokens.

5) Test predictions

  • Free: /api/predict/free?session=9am
  • Premium: /api/predict/premium?session=9am (requires JWT + premium entitlement)

6) Start mobile app

cd mobile
npm install
npx expo start
  • Use Android emulator URL: http://10.0.2.2:8000 if needed.
  • For physical device, use your PC LAN IP in EXPO_PUBLIC_API_URL.

7) Run test suites

cd backend
pytest

cd ..\mobile
npm test
npx tsc --noEmit

Backend tests currently run against in-memory SQLite for convenience.

8) Production readiness checklist

  • Replace console OTP with Twilio/Vonage/Firebase provider.
  • Wire real PayMongo checkout metadata user_id so webhook grants premium.
  • Add Detox/Appium E2E pipeline in CI.
  • Deploy backend with backend/Dockerfile and backend/cloudbuild.yaml.
  • Configure Fastlane/EAS signing and release tracks.
